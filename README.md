# Resistor Label Generator & Visualizer

A Python tool for calculating resistor values and generating industrial-style component labels.

**Main Interface**

<img src="image.png" width="400" alt="Main Interface">

## Features

* **Multi-Band Support**: Calculation and rendering for 4-band, 5-band, and 6-band (PPM) resistors.
* **Industrial Label Generation**: Generates 1600x300 resolution labels including:
    * Resistance value with auto-prefixing (Œ©, kŒ©, MŒ©, GŒ©).
    * Multiplier.
    * Tolerance percentage.
    * Wattage rating.
    * Temperature coefficient (PPM) for 6-band components.
* **Export to PNG**: Optimized for physical labeling and component storage documentation.
